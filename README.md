## Descrição

O Borland Database Engine (BDE) é o principal mecanismo de banco de dados baseado em Windows e o software de conectividade por trás do Borland Delphi, C++Builder, IntraBuilder, Paradox for Windows e Visual dBASE for Windows.

Borland Delphi é um ambiente de desenvolvimento Object Pascal baseado em Windows. Ele foi baseado no produto Borland Pascal anterior e adiciona um GUI IDE voltado para o desenvolvimento rápido de aplicativos. Algumas de suas funcionalidades foram incorporadas ao Borland C++ Builder, mas ambos continuaram a ser vendidos lado a lado. As versões atuais com suporte são vendidas pela [Embarcadero Technologies](https://www.embarcadero.com).

Este é um software de abandonado e não tenho nenhum direito sobre isso.
Este release é uma cópia feita do [Borland Delphi 7.0](https://winworldpc.com/product/delphi/70).
