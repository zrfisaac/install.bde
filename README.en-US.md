## Description

Borland Database Engine (BDE) is the Windows-based core database engine and connectivity software behind Borland Delphi, C++Builder, IntraBuilder, Paradox for Windows, and Visual dBASE for Windows.

Borland Delphi is a Windows based Object Pascal development environment. It was based on the earlier Borland Pascal product and adds a GUI IDE geared towards Rapid Application Development. Some of its functionality was merged in to Borland C++ Builder but both continued to be sold alongside each other. Current supported versions are sold by [Embarcadero Technologies](https://www.embarcadero.com).

This is an abandonware software and I don't have any right about it.
